System Manual v1.0

1. Parent Directory

The Parent Directory is the foundation of NestMind. Before any student can be enrolled, a valid Parent/Guardian record must exist in the system to link for billing and communication.

●

Identity Lock: Verify your branch name is displayed in the Identity Lock card before adding new records.

●

Add New: Clicking + Add New Parent will open the registration modal where you enter primary contact details.

2. Enrollment Process

Step 1

Select Student

Locate the child in the student dropdown. New students must be added to the Student Roster before they appear here.

Step 2

Assign Session

Assign the student to a MWF or TThS block. Ensure the session time matches the parent's preference.

Step 3

Finalize

Saving the enrollment record activates the student for the current 4-week cycle and generates the class list.
