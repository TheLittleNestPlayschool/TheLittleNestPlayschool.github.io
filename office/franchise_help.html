1. Parent Directory

The Parent Directory is the foundation of NestMind. Before any student can be enrolled, a valid Parent/Guardian record must exist.

●

Identity Lock: Always ensure your branch name is visible in the top-left Identity Lock before adding records.

●

Add New: Use the + Add New Parent button to trigger the registration modal.

2. Enrollment Process

Step 1

Select Student

Find the child in the dropdown list. If they aren't there, register them in the Student Roster first.

Step 2

Assign Session

Choose the MWF or TThS block. The system will auto-calculate the 4-week cycle.

Step 3

Finalize

Once saved, the student status will transition to "Active" in the master roster.
