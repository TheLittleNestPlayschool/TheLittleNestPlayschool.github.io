Enrollment Master Workflow

STEP 1

Parent Profile

This step establishes the primary point of contact. Use Toggle Search to find existing parents or Create Mode to enter new records.

New Records: Enter FN, LN, Phone, and Email. Data is standardized to Proper Case automatically.

Modifications: Use the Modify pill to unlock and edit contact details if an error was made during entry.

STEP 2

Student Profile

Link the child to the confirmed Parent record. You must provide the full legal name and Date of Birth.

Linked Students: When using search mode, the system only shows children currently linked to the selected parent.

Deleting/Removing: To remove a student from the workflow, you must go back to the Student Roster module.

STEP 3

Nest Assessment

Conduct the initial student evaluation. Select the evaluator from the Teacher dropdown.

  • ● Select questions via checkbox to highlight row.
  • ● Choose specific answers for each behavioral item.
  • ● Counter must reflect items chosen (Standard is 1-10).
STEP 4

Academic Session

Assign the student to their specific schedule slot. These are pre-populated by Head Office.

Warning: Ensure session times do not conflict with parent availability before locking.

STEP 5

Pickup Security

Register authorized persons for student release. Enter the name and phone number for each individual.

  • 1. Enter details ➔ 2. Click + Add Person ➔ 3. Repeat as needed ➔ 4. Finalize.
STEP 6

Finalize Sync

Set the Enrollment Date (date signed) and the School Start Date. Clicking "Save Enrollment Record" executes the final database push across all tables.

Maintenance Modules

Parents & Students Roster

Used for Maintenance only. Access these modules to update existing student statuses, edit guardian addresses, or deactivate inactive accounts.

Note: Do not use for new enrollments.

Teacher Management

A one-time setup for new hires. You will enter the teacher's full contact info, hiring date, and set their Daily Pay rate.

Actions: Edit Profile / Toggle Active Status

Session Master

This module is managed by Head Office. Franchisees have read-only access to verify active MWF and TThS time blocks for their specific location.

Resource Vault

Direct access to the 120+ Session curriculum, Summer program materials, and printable HR/Operational forms.
