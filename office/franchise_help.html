How it Works

The Enrollment Master uses a Gated UI. Steps are grayscale and locked until the previous step is confirmed. Once a step is saved, it collapses into a "Profile View" to keep the workspace clean.

STEP 0

Branch Initialization

For Head Office users, select the target branch from the dropdown. For Franchisees, this step is automatically bypassed as your Identity Lock is pre-set.

STEP 1 & 2

Parent & Student Profiles

Dual-Mode Entry

You can switch between Create Mode (for new families) and Search Mode (for existing families) using the Toggle button.

Developer Note

The system uses "Proper Case" standardization. Your text will automatically format (e.g., "dave" to "Dave") upon leaving the field.

STEP 3

Nest Assessment Evaluation

Select the evaluating teacher and date. Choose a minimum of 1-10 behavioral or academic items from the grid. These are saved individually to the database for progress tracking.

Counter Tracking

"Items: X / 10" updates in real-time as you select questions.

STEP 5

Pickup Security Protocols

Add authorized persons who may pick up the child. These are stored in the Media Vault Uploader system for verification.

Add as many as required before clicking "Finalize List".

STEP 6

Final Database Sync

Multi-Table Patching

When you click "Save Enrollment Record", the system performs three concurrent actions:

  • 1. PATCHES Student Session ID
  • 2. POSTS Session Enrollment History
  • 3. REGISTERS Pickup Protocols
